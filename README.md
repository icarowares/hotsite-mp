# CIOT para Todos - NDD Cargo

Hotsite estatico com orientacoes sobre emissao de CIOT, mudancas regulatorias,
rejeicoes frequentes e perguntas frequentes.

## Executar localmente

Abra `index.html` diretamente no navegador ou inicie um servidor estatico:

```sh
python3 -m http.server 3000
```

Em seguida, acesse `http://localhost:3000`.

## Publicacao

Este projeto e uma pagina estatica pronta para publicacao na Vercel. O arquivo
de entrada e `index.html` e nao requer etapa de build.
